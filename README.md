# React-template
Simple configuration with hot-reload and webpack v2

### Install
````bash
npm install
````

### Run Dev
````bash
npm start
````
Open [http://localhost:3000](http://localhost:3000)


#### Validate
Checks syntax errors, validate webpack.config

````bash
npm run validate
````

#### Tests
````bash
npm test
````

#### Options
This repo use pre-commit hook.
